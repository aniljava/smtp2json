smtp2json
=========

Inbound SMTP Server - Forwards parsed JSON over HTTP

### Features

- Standalone
- TLS Support
- Email parsing to extract txt/plain, txt/html and binary attachments
- Simple domain based email filtering


### Installation

    java -jar smtp2json-1.0-jar-with-dependencies.jar
 
*from source*
 
    mvn package
